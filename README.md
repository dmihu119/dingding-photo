# Lethe.丁叮｜摄影预约网站

## 本地预览
双击 `index.html` 即可预览。

## 上传 GitHub
1. 解压本压缩包。
2. 打开 GitHub 仓库 `dingding-photo`。
3. 点击 `Add file` → `Upload files`。
4. 上传：`index.html`、`style.css`、`images` 文件夹。
5. 点击 `Commit changes`。

Vercel 会自动重新部署，通常 30 秒左右生效。
